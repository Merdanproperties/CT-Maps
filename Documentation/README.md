## Directory Structure

### `guides/`
How-to guides and operational documentation:
- **BRIDGEPORT_IMPORT_GUIDE.md** - Guide for importing Bridgeport data
- **QUICK_CHANGE_GUIDE.md** - Quick guide for making changes

- **AUTO_FIX_SYSTEM.md** - Documentation for the auto-fix system
- **AUTO_RECOVERY_GUIDE.md** - Guide for automatic recovery
- **ALWAYS_RUNNING_GUIDE.md** - Guide for keeping services running
- **CHANGE_SAFETY_CHECKLIST.md** - Safety checklist for making changes
- **DEVELOPMENT_SAFETY.md** - Safety guidelines for development
- **SAFE_CHANGES_SUMMARY.md** - Summary of safe changes
- **IMPLEMENTATION_SUMMARY.md** - Implementation summary

### `troubleshooting/`
Troubleshooting guides and solutions:
- **TROUBLESHOOTING_GUIDE.md** - General troubleshooting guide
- **DEBUG_FIXES.md** - Debug fixes and solutions
- **GEOCODING_SOLUTIONS.md** - Geocoding-related solutions
- **MISSING_DATA_ANALYSIS.md** - Analysis of missing data issues
- **SEARCH_NAVIGATION_FIX.md** - Search and navigation fixes
- **NOMINATIM_FREE_SOLUTION.md** - Free Nominatim solution
- **WHY_LOCAL_NOMINATIM.md** - Explanation of local Nominatim setup

Reference docs for past fixes and patterns:
- **REFERENCE_MAP_VIEW_AND_SIDEBAR_FIXES.md** - Map view layout, sidebar whitespace/overlap, map resize (F12/DevTools), and search-bar behavior

### `setup/`
Setup and installation guides:
- **NOMINATIM_SETUP.md** - Nominatim setup instructions
- **GITHUB_SETUP.md** - GitHub repository setup
- **START_BACKEND.md** - Backend startup guide
